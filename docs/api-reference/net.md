# Networking

Modules: `net/poll`, `net/tcp`

Networking modules provide polling and TCP wrappers.

| Type | Module | Purpose |
|------|--------|---------|
| `PollEvent` | `net/poll` | Event returned from polling |
| `Poll` | `net/poll` | Poller handle |
| `TcpListener` | `net/tcp` | Listening socket wrapper |
| `TcpStream` | `net/tcp` | Connected stream wrapper |
